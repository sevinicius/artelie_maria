# Ateliê Maria Clara Fernandes

Website do Ateliê Maria Clara Fernandes - Tiaras Artesanais Exclusivas

## Sobre

Este é o site oficial do Ateliê Maria Clara Fernandes, especializado em tiaras artesanais feitas sob medida.

## Estrutura do Projeto

- `index.html` - Página principal do website
- `styles.css` - Estilos e design responsivo
- `images/` - Diretório para imagens das tiaras

## Como Visualizar

Basta abrir o arquivo `index.html` em qualquer navegador web moderno.

## Características

- Design responsivo (mobile-friendly)
- Seções: Início, Sobre, Galeria, Contato
- Interface elegante e profissional
- Otimizado para SEO

## Tecnologias

- HTML5
- CSS3
- Design Responsivo
