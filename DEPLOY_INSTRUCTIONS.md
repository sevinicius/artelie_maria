# Instruções para Publicar no GitHub Pages

## Passo 1: Criar o Repositório no GitHub

1. Acesse https://github.com/new
2. Nome do repositório: `atelie-maria-clara`
3. Deixe como **público**
4. **NÃO** marque nenhuma opção (sem README, .gitignore ou licença)
5. Clique em **"Create repository"**

## Passo 2: Conectar e Fazer Push

Execute os seguintes comandos no terminal (substitua SEU_USERNAME pelo seu nome de usuário do GitHub):

```bash
cd "c:\Users\ovini\OneDrive\Documentos\maria clara"
git remote add origin https://github.com/SEU_USERNAME/atelie-maria-clara.git
git push -u origin main
```

## Passo 3: Ativar o GitHub Pages

1. Acesse o repositório no GitHub
2. Vá em **Settings** (Configurações)
3. No menu lateral, clique em **Pages**
4. Em **Source**, selecione **"main"** branch
5. Clique em **Save**
6. Aguarde alguns minutos para o site ser publicado

## Passo 4: Acessar seu Site

O site estará disponível em:
`https://SEU_USERNAME.github.io/atelie-maria-clara/`

⚠️ **Nota**: Pode levar alguns minutos para o site ficar online após ativar o GitHub Pages.
